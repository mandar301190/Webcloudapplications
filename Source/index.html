MANDAR

  1. To create repository in GitHub , firstly have to make account on GitHub by signing with username and password. It will ask for Email ID which user wants to assign.
  1. Once the account has been set up, then it will ask to start new project . So by cliking on New project , operation of repository creation will begin.
  1. Firstly, enter the name of repository you wish to then select public option which means anyone around the globe can access your committed code.
  1. Once you click on create repository then respective repository will be displayed on your homepage with bydefault README.md file present.
  1. To clone your repository with local machine , you have to copy the link which present under clone tab on your homepage.

  • Account type should be public not private
  • Account should clone to local machine
  1. User can access multiple codes around the world
  2. Easy to use
